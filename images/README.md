# Images Directory

Add these assets for the website:

## Required Images

1. **app-icon.png** (180x180) - App icon for favicon and branding
2. **favicon-32x32.png** (32x32) - Browser favicon  
3. **favicon-16x16.png** (16x16) - Small favicon
4. **app-screenshot-hero.png** - Main hero screenshot for landing page
5. **screenshot-01.png** through **screenshot-05.png** - App Store screenshots
6. **app-store-badge.svg** - Official Apple App Store download badge

## Sources

- App screenshots: Copy from `~/projects/mydailytally-ios/screenshots/`
- App icon: Extract from iOS project
- App Store badge: Download from Apple's marketing resources

## Notes

- Use retina-ready images (2x resolution)
- Optimize for web (compress without quality loss)
- Maintain aspect ratios for screenshots
- Use PNG for screenshots, SVG for badges when possible
